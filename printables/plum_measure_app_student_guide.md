# 홍매화 측정 앱 사용법

## 1. 접속하기

선생님이 안내한 주소로 접속합니다.

- 같은 와이파이: `http://192.168.xxx.xxx:5174`
- 외부 접속 주소: `https://....trycloudflare.com`

주소가 열리지 않으면 선생님에게 새 주소를 확인합니다.

주의:

- `연구DB`와 `연구앱`은 선생님 전용 메뉴입니다.
- 학생은 `연구DB`와 `연구앱`을 누르지 않습니다.
- 잘못 눌렀다면 바로 뒤로 가거나 선생님에게 알립니다.

## 2. 측정 전 입력 정보 확인

측정하기 전에 화면 위쪽의 연구 정보를 먼저 맞춥니다.

1. 참여자ID를 입력합니다.
2. 참여자구분을 선택합니다.
3. 수목구분을 선택합니다.
   - Casing 1Year
   - Casing 2Year
   - Control Tree
4. 수목ID를 선택합니다.
   - 01부터 10까지
5. 실험회차를 확인합니다.

수목에 붙어 있는 마커 라벨과 앱의 수목ID가 같아야 합니다.

## 3. 줄기직경 측정

1. `줄기직경`을 누릅니다.
2. 수목 옆에 붙은 ArUco 마커가 화면에 보이게 촬영합니다.
3. 마커와 줄기가 흔들리지 않게 선명하게 찍습니다.
4. 사진 위에서 줄기 양쪽 끝을 터치합니다.
5. 측정값을 확인한 뒤 저장합니다.

주의:

- 마커의 검은 정사각형 한 변은 40mm입니다.
- 마커가 기울어지거나 흐리면 측정값이 부정확할 수 있습니다.
- 마커와 줄기가 같은 거리에서 보이도록 찍습니다.

## 4. 캘리퍼스 측정

1. `캘리퍼스`를 누릅니다.
2. 캘리퍼스로 같은 수목의 줄기직경을 잽니다.
3. 측정값을 mm 단위로 입력합니다.
4. 필요하면 캘리퍼스 화면 사진을 함께 저장합니다.
5. 저장합니다.

캘리퍼스 값은 카메라 측정값과 비교할 기준값입니다.

## 5. 토양측정

토양측정은 세 가지 방법 중 하나를 선택할 수 있습니다.

- 센서 CSV 불러오기
- 촬영
- 사진 불러오기

CSV를 불러올 때도 현재 선택된 수목ID로 저장됩니다.  
파일 이름이 아니라 앱에서 선택한 수목 정보가 중요합니다.

저장되는 주요 항목:

- 토양온도
- 토양수분
- 토양pH
- EC
- 비옥도
- 일조

## 6. 작업일지

1. `작업일지`를 누릅니다.
2. 촬영 또는 음성기록을 선택합니다.
3. 수목 상태나 작업 내용을 말합니다.
4. 저장 전 문장을 확인하고 필요하면 수정합니다.
5. 저장합니다.

예시:

- 가지에 병반이 보여 표시했다.
- 줄기 주변 토양 상태를 확인했다.
- 측정 위치를 다시 잡았다.

## 7. 기록 확인

측정 후에는 `이력` 메뉴에서 내가 저장한 자료가 들어갔는지 확인합니다.

확인할 내용:

- 참여자ID가 맞는지
- 수목구분이 맞는지
- 수목ID가 맞는지
- 측정값이 저장되어 있는지
- 작업일지나 사진이 필요한 경우 함께 저장되어 있는지

수목ID를 잘못 선택하고 저장했거나 값이 이상하면 바로 선생님에게 알립니다.

## 8. 저장 확인

측정 후에는 저장 완료 화면이 나오는지 확인합니다.

저장이 끝나면 다음 수목으로 이동하기 전에:

1. 수목구분 확인
2. 수목ID 확인
3. 다음 측정 시작

## 9. 자주 생기는 실수

- 수목ID를 바꾸지 않고 다음 나무를 측정함
- 마커가 사진에 일부만 보임
- 마커가 너무 작거나 흐리게 찍힘
- 캘리퍼스 값을 cm로 입력함
- 저장 버튼을 누르지 않고 화면을 나감
- 연구DB 또는 연구앱을 누름

## 10. 오늘의 측정 순서

1. 수목ID 확인
2. 줄기직경 카메라 측정
3. 캘리퍼스 측정
4. 토양측정
5. 필요한 경우 작업일지 기록
6. 이력에서 저장 확인
7. 다음 수목으로 이동

## 11. 선생님에게 알려야 하는 경우

- 앱이 열리지 않을 때
- 카메라가 켜지지 않을 때
- 마커가 인식되지 않을 때
- 저장 완료가 뜨지 않을 때
- 수목ID를 잘못 선택하고 저장했을 때
